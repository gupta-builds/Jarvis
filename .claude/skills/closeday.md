# closeday

**Description:** Create an end-of-day summary note in `60_Claude/50_Reviews/`.

**Usage:** `/closeday`

---

## Instructions

When this skill is invoked:

### 1. Gather Today's Activity

**From session logs:**
- Read `60_Claude/10_Session_Logs/log.md`
- Extract entries from today

**From file system:**
- Find notes created today (`60_Claude/`)
- Find notes modified today (all vault)

**From projects:**
- Check `20_Progress/` for completed actions
- Check off any `next:` actions that were done

**From inbox:**
- Check `00_Inbox/` for processed items

### 2. Ask User (Optional)

If user is interactive:
- "What's your energy level looking back?"
- "Anything significant I should capture?"
- "What's the one thing you're proud of today?"

### 3. Create Daily Summary

Create `60_Claude/50_Reviews/Closeday - YYYY-MM-DD.md`:

```markdown
---
type: review
status: complete
created: YYYY-MM-DD
day_of_week: [Monday-Sunday]
tags:
  - review
  - daily
notes:
  - "[[Weekly Review YYYY-WXX]]"
---

# Closeday — YYYY-MM-DD

## Summary

[2-3 sentences capturing the day's essence]

## Completed

### Projects
- [[Project 1]] — [what was done]
- [[Project 2]] — [what was done]

### Maintenance
- [ ] Inbox processed
- [ ] Clippings ingested
- [ ] Notes filed

### Learning
- [Concept 1] — [insight]
- [Concept 2] — [insight]

## Blocked / Deferred

- [Thing that didn't happen] — [reason]
- [Thing that didn't happen] — [reason]

## Insights

- [Insight 1]
- [Insight 2]

## Energy & Focus

| Metric | Rating | Notes |
|--------|--------|-------|
| Energy | X/5 | [context] |
| Focus | X/5 | [context] |
| Mood | X/5 | [context] |

## Tomorrow's Preview

- [Priority 1]
- [Priority 2]

## Gratitude / Wins

- [Win 1]
- [Win 2]

---

*Generated by /closeday*
```

### 4. Update Weekly Review

Find or create the current week's review in `60_Claude/50_Reviews/Weekly Review YYYY-WXX.md`:

- Append today's summary as a section
- Update running totals (projects touched, notes created)

### 5. Log the Session

Append to `60_Claude/10_Session_Logs/log.md`:

```markdown
## [YYYY-MM-DD] closeday | Daily summary

- Created: [[Closeday - YYYY-MM-DD]]
- Updated: [[Weekly Review YYYY-WXX]]
- Projects touched: X
- Notes created: Y
```
