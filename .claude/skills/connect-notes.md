# connect-notes

**Usage:** `/connect-notes` (prompts for notes) or `/connect-notes "Note 1" "Note 2"`

---

## Instructions

When this skill is invoked:

### 1. Get Input

Ask user:
- "Which notes should I analyze for connections?"
- Or: "What topic/project should I find connections for?"

### 2. Read the Notes

- Read each specified note fully
- Extract key concepts, entities, claims
- Note the type (evergreen, input, project, concept)

### 3. Search for Related Content

Use MCP search to find:

- Shared keywords
- Shared entities (people, orgs, tools)
- Shared projects
- Notes that link to both
- Notes in same folder/category

### 4. Analyze Connections

For each potential connection, identify:

**Type of Connection:**
- **Supports** — Note A provides evidence for Note B's claim
- **Contradicts** — Note A conflicts with Note B (flag for review)
- **Elaborates** — Note A expands on a concept in Note B
- **Precedes** — Note A is prerequisite knowledge for Note B
- **Applies** — Note A applies concept from Note B
- **Sources** — Note A is a source for Note B

**Strength:**
- **Strong** — Direct, obvious connection
- **Medium** — Indirect but meaningful
- **Weak** — Tangential; may not be worth linking

### 5. Create Connection Report

Create/update `60_Claude/40_Project_Briefs/Connections - [Topic/Date].md`:

```markdown
---
type: evergreen
status: sprout
created: YYYY-MM-DD
tags:
  - connections
  - topic-map
notes:
  - "[[Note 1]]"
  - "[[Note 2]]"
---

# Connection Analysis

## Notes Analyzed

- [[Note 1]] — [brief description]
- [[Note 2]] — [brief description]

## Strong Connections

| From | To | Type | Rationale |
|------|-----|------|-----------|
| [[Note A]] | [[Note B]] | Supports | A provides data for B's claim |

## Suggested Backlinks

### For [[Note A]]

Add to `## Connections` or `## Related`:
- [[Note B]] — [why it's related]

### For [[Note B]]

Add to `## Sources` or `## See Also`:
- [[Note A]] — [why it's related]

## Medium Connections

[Same format, lower confidence]

## Weak/Tangential

[Might not be worth linking]

## Contradictions to Review

| Note A | Note B | Nature of Tension |
|--------|--------|-------------------|
| [[A]] | [[B]] | [description] |

## Orphan Notes

Notes with no inbound links that should have some:
- [[Orphan 1]] — should link from [[X]], [[Y]]

---

*Generated by /connect-notes*
```

### 6. Offer to Apply

Ask user:
- "Should I add these backlinks now?"
- "Any connections to skip?"

If approved:
- Edit notes to add `notes:` frontmatter entries
- Add `## Connections` sections if needed
- Keep changes minimal and surgical
