# trace-topic

**Description:** Trace an idea/topic across the vault — surface all mentions, connections, and gaps.

**Usage:** `/trace-topic "topic name"`

---

## Instructions

When this skill is invoked:

### 1. Search the Vault

Use MCP search to find:

- Exact matches for the topic name
- Related concepts (synonyms, broader/narrower terms)
- Mentions in passing (footnotes, links, tables)

Search locations:
- `60_Claude/` — Distilled knowledge
- `40_Resources/` — Reference material
- `20_Progress/` — Active projects
- `10_UMN/` — Course material
- `50_Archive/` — Historical context
- `60_Claude/05_Clippings/` — Raw sources

### 2. Map the Territory

Build a map of:

**Direct Notes:**
- Notes where this topic is the main subject

**Mentions:**
- Notes where topic appears in passing
- Quotes or data points

**Connections:**
- Related topics that appear together
- Parent concepts (broader)
- Child concepts (narrower)

**Gaps:**
- Questions mentioned but unanswered
- Claims without citations
- Entities without pages

### 3. Create Trace Report

Create/update `60_Claude/40_Project_Briefs/Trace - [Topic].md`:

```markdown
---
type: concept
status: sprout
created: YYYY-MM-DD
tags:
  - trace
  - topic-map
notes:
  - "[[Related Note 1]]"
  - "[[Related Note 2]]"
---

# Topic Trace: [Topic]

## Overview

[Topic] appears in **X notes** across the vault.

## Direct Coverage

| Note | Type | Depth |
|------|------|-------|
| [[Note 1]] | evergreen | Full treatment |
| [[Note 2]] | input | Source summary |

## Mentions

| Note | Context |
|------|---------|
| [[Note 3]] | "Quote mentioning topic" |
| [[Note 4]] | Listed in table |

## Connections

**Broader:**
- [[Parent Concept]]

**Related:**
- [[Sibling 1]]
- [[Sibling 2]]

**Narrower:**
- [[Child 1]]
- [[Child 2]]

## Gaps

- [ ] Question raised in [[Note X]] — no answer yet
- [ ] Claim in [[Note Y]] — needs citation
- [ ] Entity mentioned — no page exists

## Timeline

| Date | Event |
|------|-------|
| YYYY-MM-DD | First mentioned in [[Note]] |
| YYYY-MM-DD | Full note created |
| YYYY-MM-DD | Updated with new info |

## Summary

[2-3 paragraph synthesis of what the vault knows about this topic]

---

*Generated by /trace-topic*
```

### 4. Suggest Actions

- Create missing concept pages
- Add cross-references
- Answer open questions
- Update stale claims
