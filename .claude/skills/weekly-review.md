# weekly-review

**Description:** Generate a weekly review based on recent vault activity.

**Usage:** `/weekly-review` (defaults to current week) or `/weekly-review "2026-W14"`

---

## Instructions

When this skill is invoked:

### 1. Determine Week

- Default: Current ISO week (e.g., 2026-W15)
- Or: User-specified week

Find/create `60_Claude/50_Reviews/Weekly Review YYYY-WXX.md`

### 2. Aggregate Daily Data

Read all `Closeday - *.md` files from the week:
- Sum completed items
- Extract insights
- Note patterns in energy/focus

### 3. Scan Project Activity

Read `20_Progress/` files:
- Which projects were touched?
- What progress was made?
- What's stalled?

### 4. Scan Claude Activity

Read `60_Claude/10_Session_Logs/log.md`:
- Extract all entries from the week
- Categorize: ingest, distill, create, review, lint

### 5. Scan Learning Activity

Read `10_UMN/` and `30_Order/Notes/`:
- What courses progressed?
- What concepts were learned?

### 6. Create Weekly Review

```markdown
---
type: review
status: complete
created: YYYY-MM-DD
week: YYYY-WXX
tags:
  - review
  - weekly
notes:
  - "[[Monthly Review YYYY-MM]]"
  - "[[Closeday - Day 1]]"
  - "[[Closeday - Day 2]]"
---

# Weekly Review — YYYY-WXX

## Summary

[3-5 sentences capturing the week's essence]

## Projects

### Progress Made

| Project | Status Change | What Happened |
|---------|---------------|---------------|
| [[Project 1]] | seed → sprout | [description] |
| [[Project 2]] | sprout → tree | [description] |

### Stalled / Attention Needed

- [[Project X]] — [why stalled, what's needed]

## Claude Activity

| Day | Session Type | Output |
|-----|--------------|--------|
| Mon | ingest | [source summary] |
| Wed | distill | [concept note] |
| Fri | review | [weekly summary] |

**Totals:**
- Notes created: X
- Sources ingested: Y
- Concepts distilled: Z

## Learning

### Courses

- [[Course 1]] — [what covered this week]
- [[Course 2]] — [what covered this week]

### Concepts Mastered

- [[Concept 1]] — [evidence of mastery]
- [[Concept 2]] — [evidence of mastery]

## Insights

- [Insight 1]
- [Insight 2]
- [Insight 3]

## Energy Patterns

| Day | Energy | Focus | Mood |
|-----|--------|-------|------|
| Mon | X/5 | X/5 | X/5 |
| Tue | X/5 | X/5 | X/5 |

**Patterns:**
- [Observation about energy/focus patterns]

## Next Week Priorities

1. [Priority 1]
2. [Priority 2]
3. [Priority 3]

## Open Questions

- [Question 1]
- [Question 2]

---

*Generated by /weekly-review*
```

### 7. Link to Monthly Review

If this is the last week of the month:
- Create or update `60_Claude/50_Reviews/Monthly Review YYYY-MM.md`
- Link this weekly review

### 8. Log the Session

Append to `60_Claude/10_Session_Logs/log.md`
