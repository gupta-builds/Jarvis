## Key Points to Know

### Primary vs. Secondary Response
| Feature | Primary Response | Secondary Response |
|---------|-----------------|-------------------|
| Exposure | First time to this antigen | Re-exposure |
| Speed | Slow | Faster |
| Antibody peak | Lower | Much higher |
| Duration | Shorter | Longer |
| Why? | Must find & clone rare matching lymphocytes | Memory cells already exist; activate immediately |

- **Three measurable differences:** speed, peak antibody concentration, duration.
- **Greater antibody production in secondary response = more B cells making antibodies** (not more per cell).

### Cross-Talk Between Innate and Adaptive
- Phagocytes (innate) engulf pathogens and **display antigen fragments** → T cells encounter and activate (adaptive).
- Helper T cells secrete **cytokines** that stimulate phagocytes (innate) → more phagocytosis.
- B cell antibodies activate **complement** (innate) → phagocytosis, inflammation, cytolysis.
- The systems are distinct but interdependent.

## Fast Recap

> Primary = slow, small, short. Secondary = fast, large, lasting.  
> Memory cells = the reason the second response is stronger.  
> Innate and adaptive constantly signal each other (cytokines, antigen presentation, antibody-complement activation).
