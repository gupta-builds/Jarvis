## Fast Review
- **SEVEN-UP**: seminiferous tubules -> epididymis -> vas deferens -> ejaculatory duct -> urethra -> penis.
- **Ovulated ova path if not fertilized**: ovary -> fallopian tube -> uterus -> cervix -> vagina -> vaginal opening.
- **Hormone specificity**: a hormone affects only target cells with the right receptor.
- **Ovulation clue**: LH surge.
- **Luteal/secretory clue**: progesterone high.
- **Cancer clue**: multiple mutations in growth-control genes, not one magic cancer switch.
- **BRCA clue**: risk goes up, but cancer is not guaranteed.
