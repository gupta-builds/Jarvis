# Meiosis and Gametogenesis

## Exam Objectives Covered
- ME1: define homologous chromosome pair.
- ME2: identify oogonium, spermatogonium, sperm, egg, fertilized egg, and somatic cell as haploid or diploid.
- CD2: compare mitosis and meiosis; identify which cells normally undergo each.
- Comparing reproduction: sperm vs ova production differences.

## 30-Second Version
Meiosis is the cell division process that makes **haploid gametes**. Sperm and eggs must be haploid so that fertilization restores the diploid chromosome number instead of doubling it every generation.

## Homologous Chromosomes
- Humans usually have 23 pairs of chromosomes.
- **Homologous pair**: two chromosomes with the same genes in the same locations.
- One chromosome in each pair is inherited from each parent.
- Homologous chromosomes can carry different alleles.

Example: both homologous chromosomes may carry an HLA gene at the same locus, but they may carry different HLA alleles.

## Haploid vs Diploid
- **Diploid (2n)**: two copies of each chromosome.
- **Haploid (n)**: one copy of each chromosome.

| Cell | Haploid or diploid? | Why |
|---|---|---|
| Somatic/body cell | Diploid | Body cells usually have 23 pairs. |
| Skin cell | Diploid | Body cell. |
| Liver cell | Diploid | Body cell. |
| Spermatogonium | Diploid | Germ-line starting cell before meiosis. |
| Oogonium | Diploid | Germ-line starting cell before meiosis. |
| Sperm | Haploid | Gamete after meiosis. |
| Egg / ovum | Haploid | Gamete before fertilization. |
| Fertilized egg / zygote | Diploid | Sperm and egg chromosomes combine. |

Exam trap: an egg is haploid **before** fertilization. A fertilized egg is diploid.

## Gametogenesis
**Gametogenesis** means production of gametes.

| Feature | Spermatogenesis | Oogenesis / ova production |
|---|---|---|
| Gonad | Testes | Ovaries |
| Starts | Puberty | Oocytes are present before birth; cycle resumes later |
| Output from one meiotic event | Usually 4 viable sperm | 1 viable egg plus polar bodies |
| Division style | Equal cytoplasm divisions | Unequal cytoplasm divisions |
| Gamete size | Small, motile, many produced | Large, nonmotile, fewer produced |
| Hormone dependence | Strongly regulated by GnRH, FSH, LH, testosterone | Regulated by GnRH, FSH, LH, estrogen, progesterone |

Lecture emphasis: males and females in this class are defined by gamete type across biology: sperm producers are male; egg/ova producers are female. Humans are more complicated socially and anatomically, but this definition helps separate gamete production from gender.

### Spermatogenesis sequence
1. Spermatogonium is diploid.
2. It divides/grows into a primary spermatocyte.
3. Meiosis I makes two secondary spermatocytes.
4. Meiosis II makes four spermatids.
5. Spermatids differentiate into sperm.

Key outcome: one diploid starting cell can produce four haploid sperm.

### Oogenesis sequence
1. Oogonium is diploid.
2. Primary oocytes are present before birth.
3. After puberty, one oocyte usually resumes development during a cycle.
4. Meiosis divisions are unequal.
5. One large egg gets most cytoplasm.
6. Small polar bodies receive extra chromosomes but usually do not become viable gametes.

Key outcome: one diploid starting cell produces one viable haploid egg plus polar bodies.

Exam trap: the difference is not only "sperm vs egg." It is also continuous vs cyclical timing, equal vs unequal cell division, and four viable gametes vs one viable gamete.

## Mitosis vs Meiosis
| Feature | Mitosis | Meiosis |
|---|---|---|
| Main job | Growth, repair, replacement, embryonic cell multiplication | Gamete production |
| Starting cell | Diploid somatic or germ-line cells depending context | Diploid germ-line cells |
| Ending cells | Usually 2 genetically similar diploid cells | Haploid gametes |
| Chromosome number | Maintained | Reduced by half |
| Genetic variation | Not the main goal | Crossing over/independent assortment create variation |
| Theme 4 connection | Cleavage, development, normal cell replacement, cancer if uncontrolled | Sperm and egg production |

Exam trap: early embryo growth uses **mitosis**, not meiosis. Meiosis makes gametes before fertilization.

## How This Connects to Development
1. Meiosis makes haploid sperm and egg.
2. Fertilization combines them into one diploid zygote.
3. Mitosis/cleavage turns the zygote into many cells.
4. Differentiation makes those cells behave differently even though they mostly share the same DNA.

## Exam-Style Applications
- "A human body cell has 23 homologous pairs. What is true of sperm?" -> sperm is haploid.
- "Are sperm genetically identical to each other?" -> no, meiosis creates variation.
- "Are gametes present before puberty?" -> sperm are not; oocytes/eggs are present before birth in the ovarian pathway.
- "Where does meiosis occur?" -> testes for sperm production, ovaries for ova production.
