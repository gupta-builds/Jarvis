## Plugins
### Dashboards
Dashboards are valuable **only if they reduce thinking + clicking**:
- “What do I need to do next?” → open dashboard, see it.
- “What’s due soon?” → open dashboard, see it.
- “What inbox notes are unprocessed?” → open dashboard, see it.
- “What did I capture from Readwise this week?” → open dashboard, see it.
So the win is: **you stop manually maintaining lists**.
### Missing
The missing piece: your “note factory” (templates + QuickAdd + Templater)
If you want “insane Obsidian” without it becoming a second job, you need 2 things:
1. **Templates that create the structure automatically**
2. **One-key capture actions** (QuickAdd) that drop notes into the right folder with the right frontmatter.
