# ⚡ Obsidian Hotkeys (Final Setup)
## 🗂️ Files / Search / Explorer
- Rename File — **F2**
- Search Current File — **Ctrl + F**
- Omnisearch (Vault Search) — **Alt + F**
- Search & Replace in Current File — **Ctrl + H**
- Search in All Files — **Ctrl + Shift + F**
- Open Quick Switcher — **Ctrl + O**
- Create New Note — **Ctrl + N**
- Create Note to the Right — **Ctrl + Shift + N**
- Create New Note from Template — **Alt + N**
- Command Palette — **Ctrl + P**
- Close Current Tab — **Ctrl + W**
- Close All Tabs — **Alt + W**
- Close Window — **Ctrl + Shift + W**
## 🔗 Links / Navigation
- Insert Markdown Link — **Ctrl + K**
- Toggle Backlinks in Document — **Alt + B**
- Excalidraw: Insert Link to File — **Ctrl + Shift + K**
## ✍️ Editing / Formatting
- Toggle Bold — **Ctrl + B**
- Toggle Italic — **Ctrl + I**
- Toggle Comment — **Ctrl + /**
- Toggle Checkbox Status — **Ctrl + L**
- Insert Code Block — **Alt + V**
- Add File Property — **Ctrl + ;**
## 🧩 Plugin Hotkeys
### 🧠 Templater
- Open Insert Template Modal — **Alt + E**
- Replace Templates in Active File — **Alt + R**
- Jump to Next Cursor Location — **Alt + Tab**
### ⚡ QuickAdd
- Run QuickAdd — **Alt + Q**
### 🗓️ Calendar
- Open Calendar View — **Alt + C**
## 🧭 Navigation / Tabs / View
- Toggle Reading View — **Ctrl + E**
- Undo Close Tab — **Ctrl + Shift + T**
- New Tab — **Ctrl + T**
- Go to Last Tab — **Ctrl + 9**

- Go to Previous Tab — **Ctrl + Alt + left**

- Go to Tab #2 — **Ctrl + 2**
- Go to Tab #3 — **Ctrl + 3**

- Go to Tab #7 — **Ctrl + 7**
- Go to Tab #8 — **Ctrl + 8**

- Navigate Forward — **Ctrl + Alt + →**
