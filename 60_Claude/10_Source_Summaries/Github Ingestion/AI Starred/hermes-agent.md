# hermes-agent — NousResearch

**Repo:** https://github.com/NousResearch/hermes-agent
**Stars:** 171,870 | **Language:** Python | **License:** MIT

## What It Is

The most-starred AI coding agent on GitHub as of May 2026. Built by Nous Research (the open-weights fine-tuning lab). Positions itself as "the agent that grows with you" — i.e., learns from your codebase and adapts over time.

## Core Capabilities

- Full ACP (Agent Communication Protocol), MCP, and Claude Code integration
- Works with Claude, OpenAI, Codex, and local models
- 14,823 open issues (large active community, rough edges)
- Python-based, cross-platform

## What Makes It Different

Nous Research is known for fine-tuned Hermes model series. This agent is built on top of those models + MCP tooling. The "grows with you" framing suggests long-term memory or skill accumulation — but the repo is massive and fast-moving; verify current state before using.

## Why It's Starred

Highest-star coding agent right now. Worth monitoring as a reference for what the community is building toward. At 171K stars with 28K forks, it's the de facto open source alternative to Claude Code.

## Caveats

- 14K open issues = high activity but rough edges
- Large, fast-moving codebase; check changelog before integrating
- Competes directly with opencode, goose, Claude Code
