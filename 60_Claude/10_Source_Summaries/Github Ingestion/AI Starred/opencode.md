# opencode — anomalyco

**Repo:** https://github.com/anomalyco/opencode
**Stars:** 166,784 | **Language:** TypeScript | **License:** MIT

## What It Is

Open source coding agent by anomalyco (SST creators). Terminal-first, multi-model, actively maintained. Competes directly with Claude Code and hermes-agent.

## Core Capabilities

- Multi-model: OpenAI, Anthropic, local models
- TypeScript + TUI (terminal UI)
- MCP support
- Dev branch = main development branch

## Architecture

Built by the team behind SST (serverless stack tooling). They understand infrastructure deeply — the agent likely has strong opinions about deployment, CI/CD, cloud workflows. TypeScript codebase means it's hackable by the JS/TS ecosystem.

## Why It's Starred

166K stars means the community sees this as a real alternative to Claude Code. Worth watching if Claude Code becomes too expensive or lock-in becomes a concern. Also good to study their CLAUDE.md / agent design for patterns to steal.
