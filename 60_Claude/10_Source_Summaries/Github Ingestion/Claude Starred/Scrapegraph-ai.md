# ScrapeGraph AI

**What it is:** A Python web scraping library that uses LLMs to extract structured data from pages — you describe what you want in natural language and it builds an extraction pipeline.

**Why it's here:** Useful for research data collection tasks where the target site has no API.

**Why it's not a priority:** This is a general-purpose scraping tool, not a Claude integration. Claude Code already has web fetch capabilities via MCP. For structured research data extraction at scale, this is worth revisiting — but it's not Claude-specific and doesn't plug into Jarvis or the agentic workflow directly.
