# Awesome Agent Skills

**What it is:** A curated list of 630+ agent skills from official dev teams (Anthropic, Vercel, Stripe, Cloudflare, Supabase, Netlify, HashiCorp, Google, etc.) and community contributors, compatible with Claude Code, Codex, Gemini CLI, and others.

**Why it's here:** One-stop index for finding official skills from companies whose tools you actually use (Supabase postgres best practices, Stripe integration patterns, Firecrawl scraping).

**Why it's not a priority:** This is a link list — the skills themselves live in their source repos. The real value is using it as a search tool when you need a skill for a specific integration, not reading it end-to-end. File it under "consult when looking for skills" rather than studying it.
