# Vibe Kanban

**What it is:** A kanban board that runs coding agents (Claude Code, Codex, Gemini CLI, etc.) in isolated git worktrees with built-in browser, diff review, and PR creation — essentially a project management UI for parallel agent workflows.

**Why it's here:** Solves a real coordination problem: managing multiple Claude Code sessions working on different branches simultaneously with a visual overview.

**Why it's not a priority:** As of the starred date, **Vibe Kanban is sunsetting** — the README prominently links a shutdown announcement. The core idea (multi-agent kanban with worktrees) is worth watching for a successor, but installing a shutting-down tool creates dependency on abandoned code.
