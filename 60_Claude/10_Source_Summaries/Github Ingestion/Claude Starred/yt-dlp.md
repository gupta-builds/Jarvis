# yt-dlp

**What it is:** A feature-rich CLI audio/video downloader supporting 1,800+ sites, forked from youtube-dl with active maintenance and a large plugin ecosystem.

**Why it's here:** Utility tool — probably starred as a go-to for downloading research videos, course content, or media for offline use.

**Why it's not a priority:** No Claude integration path. Useful as a CLI tool but unrelated to Claude workflows, agentic systems, or Jarvis. If needed, just `pip install yt-dlp` and use it directly.
