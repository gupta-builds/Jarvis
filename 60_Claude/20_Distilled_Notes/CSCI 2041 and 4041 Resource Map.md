# CSCI 2041 and 4041 Resource Map

The two course-resource clusters serve different jobs. `CSCI 2041` points to official language documentation for execution details, while `CSCI 4041` points to solutions, setup help, and full course mirrors that support deeper study outside the weekly notes.

## Split by course
- `CSCI 2041`: use official OCaml references when syntax, semantics, or standard workflow details need a stable answer.
- `CSCI 4041`: use the algorithms cluster when the textbook, homework, or weekly note needs a second source, setup help, or a full course archive.

## Working rule
- Weekly notes still stay primary inside the vault.
- External links are support material, not replacements for the course notes.
- Shared canonical source notes make it easier to keep `DSA` and `Introduction to Algorithms` synced around the same external references.
